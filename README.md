# Workflow

Just documenting some of the stuff I use for my Mac workflow.

### Core Tools

**Aerospace**

[https://github.com/nikitabobko/AeroSpace](https://github.com/nikitabobko/AeroSpace)

An amazing window tiling manager.

**Raycast**

[https://www.raycast.com](https://www.raycast.com)

The better Spotlight Search.

- Settings: rebind Raycast to CMD + Space and unbind Spotlight Search default CMD + Space (System Settings)

**Homerow**

[https://www.homerow.app](https://www.homerow.app)

Keyboard shortcuts for buttons and Vim-style scroll.

**Superwhisper**

[https://superwhisper.com](https://superwhisper.com)

Free voice-to-text for every app.

### Dev Tools

**Kitty**

See `terminal-setup-guide` for full information for setup

**Neovim**

[https://neovim.io](https://neovim.io)

Vim but better (see init.lua).

**Visual Studio Code**

[https://code.visualstudio.com](https://code.visualstudio.com)

Default code editor.

- Settings: View > Appearance > Activity Bar Position > Top, One Dark Pro Night Flat theme

- Extensions: Vim, Rainbow CSV, Remote - SSH, Remote Explorer, Remote - SSH Editing Configuration Files, vscode-pdf, Auto Rename Tag, Better Comments, Container Tools, Dev Containers, Docker, Docker DX, gitignore, Material Icon Theme

**Xcode**

[https://developer.apple.com/xcode/](https://developer.apple.com/xcode/)

For iOS dev.

- Extensions: Vim text settings

**Android Studio**

[https://developer.android.com/studio](https://developer.android.com/studio)

For Android and KMP dev.

- Plugins: Kotlin Multiplatform, IdeaVim

**IntelliJ**

[https://www.jetbrains.com/idea/](https://www.jetbrains.com/idea/)

For Java dev.

- Plugins: IdeaVim
